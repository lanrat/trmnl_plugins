# [Pearls Before Swine Comic](https://trmnl.com/recipes/248035?ref=mrlanrat)

Displays the daily [Pearls Before Swine](https://www.gocomics.com/pearlsbeforeswine) comic by Stephan Pastis on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest Pearls Before Swine comic strip.

## Settings

No custom settings required.
