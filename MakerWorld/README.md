# [MakerWorld Trending Models](https://trmnl.com/recipes/272038?ref=mrlanrat)

Show trending 3D printing projects from [MakerWorld](https://makerworld.com/) on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin displays trending models from MakerWorld, Bambu Lab's 3D model sharing platform.

## Settings

No custom settings required.
