# [Doonesbury Comic](https://trmnl.com/recipes/248037?ref=mrlanrat)

Displays the daily [Doonesbury](https://www.gocomics.com/doonesbury) comic by Garry Trudeau on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest Doonesbury comic strip.

## Settings

No custom settings required.
