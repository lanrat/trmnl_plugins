# [Luann Comic](https://trmnl.com/recipes/248034?ref=mrlanrat)

Displays the daily [Luann](https://www.gocomics.com/luann) comic by Greg Evans and Karen Evans on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest Luann comic strip.

## Settings

No custom settings required.
