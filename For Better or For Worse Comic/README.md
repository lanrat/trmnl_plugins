# [For Better or For Worse Comic](https://trmnl.com/recipes/248038?ref=mrlanrat)

Displays the daily [For Better or For Worse](https://www.gocomics.com/forbetterorforworse) comic by Lynn Johnston on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest For Better or For Worse comic strip.

## Settings

No custom settings required.
