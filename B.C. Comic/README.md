# [B.C. Comic](https://trmnl.com/recipes/248039?ref=mrlanrat)

Displays the daily [B.C.](https://www.comicskingdom.com/bc) comic by Johnny Hart on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest B.C. comic strip.

## Settings

No custom settings required.
