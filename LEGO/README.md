# [Random LEGO](https://trmnl.com/recipes/269418?ref=mrlanrat)

Display a random LEGO Minifig or Set from [Rebrickable](https://rebrickable.com/) on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin shows a random LEGO set or minifig from the Rebrickable catalog, including its image and details.

## Settings

- **Catalog**: Choose whether to display random Sets or Minifigs
