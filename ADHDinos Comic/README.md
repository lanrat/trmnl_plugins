# [ADHDinos Comic](https://trmnl.com/recipes/248040?ref=mrlanrat)

Displays the daily [ADHDinos](https://www.adhdinoscomic.com/) comic by Ryan Keats on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest ADHDinos comic strip, featuring dinosaurs navigating life with ADHD.

## Settings

No custom settings required.
