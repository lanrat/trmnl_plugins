# [Non Sequitur Comic](https://trmnl.com/recipes/248036?ref=mrlanrat)

Displays the daily [Non Sequitur](https://www.gocomics.com/nonsequitur) comic by Wiley Miller on your TRMNL device.

![Screenshot](screenshot.png)

## Description

This plugin fetches and displays the latest Non Sequitur comic strip.

## Settings

No custom settings required.
